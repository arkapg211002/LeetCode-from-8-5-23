## Question

Given two integers `left` and `right` that represent the range `[left, right]`, return the bitwise AND of all numbers in this range, inclusive.

## Level : Medium

This problem is of medium difficulty.

## Table for Input Output

| Input | Output |
| ----- | ------ |
| 5, 7  | 4      |
| 0, 0  | 0      |
| 1, 2147483647 | 0 |

## Constraints

- 0 <= left <= right <= 2^31 - 1
